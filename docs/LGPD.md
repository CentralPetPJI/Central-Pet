# LGPD - Central-Pet

## Retenção de Dados

Em conformidade com a Lei Geral de Proteção de Dados (LGPD) e para fins de auditoria e segurança, a plataforma Central-Pet adota as seguintes políticas de retenção:

1. **Desativação de Conta**: Quando um usuário solicita a desativação de sua conta, os dados não são excluídos imediatamente.
2. **Período de Retenção**: Os dados pessoais e registros de atividades são mantidos por um período de **90 dias** após a desativação.
3. **Propósito**: Este período é necessário para garantir a integridade de auditorias em processos de adoção em andamento ou recentemente concluídos, bem como para cumprir obrigações legais.
4. **Exclusão Permanente**: Após o prazo de 90 dias, os dados são removidos permanentemente de nossos sistemas ativos.
5. **Reativação**: Durante o período de 90 dias, o usuário pode solicitar a reativação da conta entrando em contato com o suporte.
